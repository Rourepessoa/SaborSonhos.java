Sabor Sonhos
Descrição
O Sabor Sonhos é uma aplicação desenvolvida em Java para o gerenciamento de produtos e pedidos. Este projeto foi criado para otimizar o fluxo de trabalho diário de um estabelecimento do ramo alimentício, facilitando o registro de vendas e o controle de estoque.

Funcionalidades
Cadastro e atualização de produtos no catálogo.

Registro de novos pedidos de clientes.

Controle de estoque de ingredientes e produtos finalizados.

Cálculo automático do valor total de vendas.

Tecnologias Utilizadas
Java: Linguagem de programação principal do projeto.

Pré-requisitos
Para executar este projeto localmente, você precisará ter instalado em sua máquina:

Java Development Kit (JDK) 8 ou superior.

Uma IDE para desenvolvimento Java (como IntelliJ IDEA, Eclipse, NetBeans) ou acesso a um terminal com as variáveis de ambiente do Java configuradas.
