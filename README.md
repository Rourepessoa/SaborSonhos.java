# SaborSonhos.java
Cardapio/Menu, de um restaurante de bolo contendo as seguintes detalhes; Exibir um menu de opções da lanchonete , Permitir ao usuário selecionar produtos, Utilizar estrutura switch case para tratar as opções, Utilizar laços de repetição para manter o sistema funcionando até o encerramento, Exibir ao final o resumo do pedido na tela.
